# Stone Photography 
## V1.0

Stone Photography is the first version of my personal photography website, created as both a portfolio project and a learning journey. The site is designed to showcase my photography in a clean, responsive format while also helping me develop my front-end skills.

Version 1.0 marks the beginning of the project, starting with a Bootstrap based layout before moving into React to support future growth, better component structure and improved maintainability.

## About the Project

Stone Photography is a beginner photography website built to display my work and introduce myself as a photographer. As someone early in my photography journey this project reflects both my creative interests and my progress in web development.

The site currently includes:

- A responsive navigation bar
- A gallery section with lightbox functionality
- An About Me section with a personal introduction and image
- A Contact Me section with social media icons
- Consistent styling and typography across the site

## Version 1.0 Features

### Initial setup
- Set up the project with Bootstrap 5.3.8
- Added SCSS utility mixins for responsive design
- Added accessibility helpers such as a visually hidden mixin
- Added RFS mixin support for responsive sizing

### Gallery
- Created the initial gallery layout
- Added a simple image lightbox feature
- Improved the gallery layout using Bootstrap components
- Added test images to help shape the design
- Updated the gallery styling and introduced the "Milky Lemon" font
- Adjusted the gallery section so it fits cleanly within its page region

### Navigation
- Improved the navbar to be more responsive
- Added better styling for navigation items
- Included colour and width changes on selection for a more interactive feel

### React migration
- Decided to move the website from the earlier structure into React
- Made this change for learning purposes and to benefit from React's component-based structure
- Removed unnecessary legacy files such as the old `script.js`

### About section
- Created `About.jsx`
- Added a personal description and background in photography
- Updated the section heading from **About** to **About Me**
- Added a personal photo and adjusted the layout using Bootstrap for smoother responsiveness across devices

### Contact section
- Completed the `Contact Me` section
- Added Bootstrap Icons for social media links
- Styled the section to match the rest of the site

### Styling and cleanup
- Adjusted fonts to improve consistency across the project
- Added a reusable subtitle style for white text where needed
- Performed general code cleanup

## Why React?

A major part of version 1.0 was deciding to move development into React. This was done not only to improve the structure of the website, but also as part of my goal to learn modern frontend development techniques.

Using React allows the project to grow more easily over time, especially as I continue to add new features and improve the design.

## Project Goals

This project is being developed to:

- Build a clean and responsive personal website
- Improve my knowledge of Bootstrap and React
